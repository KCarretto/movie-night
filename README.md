# Plot Polls

Plot Polls is a serverless, peer-to-peer movie voting app that runs as a single
static page (`index.html`).

## What it does

- Create a room and invite friends with a join link or QR code
- Nominate movies and vote in ranked order
- Pick a winner from group results
- Save personal watch history and ratings locally
- Generate recommendations from shared taste vectors

## Tech stack

- Plain HTML/CSS/JavaScript (no build step)
- WebRTC networking via PeerJS
- Tailwind CSS from CDN
- QR generation/scanning via qrcode.js and html5-qrcode
- Protobuf catalog (`movies.pbf`) + binary embeddings (`embeddings.bin`)

## Run locally

This project is static. To run it, serve the repository root with any static
HTTP server and open it in a browser.

Example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Data pipeline

Movie catalog and embedding files are generated by scripts in `scripts/` and
updated by `.github/workflows/sync-movies.yml`.

See `scripts/README.md` for details on regenerating `movies.pbf` and
`embeddings.bin`.
